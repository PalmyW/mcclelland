# Club AFL Rank AFLW Rank Sum / Points Tie‑break

Data sources

  1. AFL ladder: Squiggle API standings (live).
  2. AFLW ladder: Local aflw.json file included in this repo.
How positions are calculated
  • Rank‑sum (official 2025+): Club score = (current AFL ladder rank) + (current AFLW ladder rank). Lowest total leads.
  • Tie‑breakers: (1) Legacy trophy points = 4 per AFL win (2 per draw), 8 per AFLW win (4 per draw). (2) Combined percentage = weighted blend of AFL and AFLW % (normalized).
  • Legacy mode: Shows the pre‑2025 aggregate points leaderboard for context.
